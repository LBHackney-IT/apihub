What is an API?

API stands for "Application Programming Interface".
They provide a set of methods or actions that enable a systems developer to communicate with another system in a controlled manner.

Why do you need an API?

It's likely that every app on your smart phone uses a API behind the scenes.
Each time you tell your smart speaker to turn on the lights in your living room a series of API calls will likely occur in the background in a controlled manner which eventually culminates in a bulb switching on.

One important part in the paragraph above is that the APIs should be device agnostic. It shouldn't matter what brand of phone or smart speaker you have or even what the device is. The API should behave in the same way to give the same end result as long as the API is visible to that device and the device is able to connect to it.
